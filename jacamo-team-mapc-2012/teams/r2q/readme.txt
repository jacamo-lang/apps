Nesse time foram implementadas as seguintes funcionalidades:
1) Ele é baseado no R2p.
2) Foi adicionado uma idéia para que os agentes consigam formar mais rápido uma área boa.
3) Tratado situação do full-map, quando os agentes dominam o mapa praticamente todo, todos os agentes devem parar. Também funções vitais como reparação, ataque, inspeção e exploração. Se tem todo mapa dominado quem precisa tomar a iniciativa é o inimigo.
